# PRO-BASKET — React + TypeScript + styled-components (Create React App)

## Jak uruchomić
```bash
npm install
npm start
```

Aplikacja wystartuje na http://localhost:3000

## Stos
- React 18
- TypeScript
- styled-components 6
- CRA (react-scripts 5)
- lucide-react (ikony)

## Struktura
- `src/App.tsx` — cała strona-wizytówka w jednym komponencie
- `src/theme.ts` — kolory/parametry
